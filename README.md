# forgemodcheatsheet
Calculates the cheapest sequence of colors to forge stuff

## Installation for Windows
The easiest way to use this tool is to download the pre-built executable:

1. Go to the [Latest Release](https://github.com/jkibort928/forgemodcheatsheet/releases/tag/v1.0.0).
2. Download `ForgingCheatSheet.exe`.
3. Double-click to run! 

*Note: If Windows shows a "Protected your PC" warning, click **More Info** -> **Run Anyway**.*
